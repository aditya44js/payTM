# payTM
A full-stack Paytm-inspired web application that allows users to securely create accounts, manage balances, transfer money, and perform peer-to-peer transactions in real time. The project focuses on secure authentication, responsive UI design, and efficient backend APIs using modern web technologies.

⭐️ Features
User Signup & Login Authentication
JWT-based Secure Authorization
Send Money Between Users
Wallet Balance Management
Search and View Users
Responsive Dashboard UI
Protected Routes
REST API Integration
MongoDB Database Connectivity

🛠 Tech Stack
#Frontend
React.js
Tailwind CSS
Axios
React Router DOM

#Backend
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
